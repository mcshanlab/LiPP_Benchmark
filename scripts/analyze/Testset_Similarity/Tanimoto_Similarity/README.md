Analyze the Tanimoto Similarity between lipids

## Environments

install the environments with `environment.yml` \

`conda activate analysis` 

## Steps to run the code

### 1. Run the similarity calculations between pairs of lipids
`python get_similarity.py`

### 2. Find the best matching lipid/protein pairs
`python find_best_match.py`

### 3. Plot the results:
`python plot_result.py`
